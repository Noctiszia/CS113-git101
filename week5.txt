1. Scope Rules 
บางบริเวณของโปรแกรมตัวแปรที่ถูกกำหนดขึ้นในบริเวณอาจเข้าถึงได้และเมื่อออกจากบริเวณดังกล่าวตัวแปรนั้นอาจไม่ส่ามารถถูกเข้าถึงได้อีก
1.1 ตัวแปร Local 
ตัวแปรที่ถูกประกาศไว้ภายใน block,ภายในตัวฟังก์ชัน รวมถึงตัวแปรพารามิเตอร์ของฟังก์ชัน จะเป็นตัวแปรlocalซึ่งสามารถถูกใช้งานได้เพียงภายในฟังก์ชันที่ถูกประกาศไว้
//ถ้าใช้คีย์เวิร์ด static ตัวแปรจะถูกเข้าถึงได้ในฟังก์ชันเท่านั้น 
//ค่าจะยังอยู่ ใน memory แม้ว่าจะออกจากฟังก์ชันไปแล้ว 
1.2 ตัวแปร Global ตัวแปรที่ถูกประกาศไว้นอกฟังก์ชันเป็นตัวแปร global ซึ่งสามารถถูกเข้าถึงได้จากฟังก์ชันต่าง ๆ รวมถึงฟังก์ชัน main ได้ 
// ถ้า ชื่อซ้ำกับ ตัวแปร local การแก้ไขค่าใน local จะไม่ส่งผลกับค่าของ global 
//ให้พยายามใช้ global ดีกว่า การนิยามฟังก์ชันและเรียกใช้ไม่จำเป็นต้องทำพร้อมกันสามารถทำแยกกันได้ และสามารถแยกอยู่คนละไฟล์ได้ 
2. Random Number Generator 
rand() เป็นฟังก์ชัน ใน C Standard Library ที่ใช้ในการสร้างเลขสุ่มจำนวนเต็มระหว่าง 0 และ RAND_MAX โดย RAND_MAX เป็นค่าคงที่ถูกกำหนดไว้ให้ใน<stdlib.h> 
//ถ้าเรียกใช้หลายครั้งค่าที่ได้จะแตกต่างกันไป
